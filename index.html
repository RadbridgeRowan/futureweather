Overview

Future Weather is a minimal, fast, and integration‑ready platform for generating climate‑adjusted weather files for building performance modeling. It morphs historical weather with future climate projections to create EPW files, with optional DDY and STAT outputs.

Built in collaboration with the academic creator of the original Future Weather Generator, our method uses IPCC AR6 (CMIP6) scenarios and has been validated against legacy outputs. Use the clean web UI for one‑off files, or integrate via API to automate your workflows.

How it works

1. Choose location and scenario

Select a site (by city, station, or lat/long), pick a target year (e.g., 2030–2080) and scenario (e.g., SSP2‑4.5 or SSP5‑8.5). Optionally choose single GCM or an ensemble.

2. Generate and quality‑check

We morph the historical weather with CMIP6 projections using a validated method. Automated QA ensures consistent ranges and metadata.

3. Download or integrate

Download EPW (and optional DDY/STAT) immediately, or pull via API into EnergyPlus/OpenStudio or your internal tools.
